# Information Admin — Panduan Mengelola Informasi & Pengumuman

Fungsi: membuat, mengedit, dan memoderasi entri informasi (pengumuman, pesan internal, paket masuk) serta meninjau antrian permintaan.

Area utama:
- Request Queue: menampilkan booking/permintaan offline & online yang perlu diproses.
- Index Table: daftar entri informasi (edit/delete, publish/unpublish).
- Create/Edit Modal: form untuk menambah atau mengubah informasi.
- Inform Modal & Reject Modal: untuk mengirim pemberitahuan atau menolak permintaan dengan alasan.

Langkah umum:
1. Buka `Information` di sidebar.
2. Di `Request Queue`, pilih item untuk dilihat atau diproses.
3. Gunakan `Create` untuk membuat pengumuman baru — isi judul, isi, tanggal, dan lampiran jika perlu.
4. Untuk permintaan yang butuh persetujuan, gunakan `Inform` untuk mengirim pemberitahuan atau `Reject` untuk menolak dengan alasan.

Catatan:
- Modal bisa ditutup dengan tombol atau tekan `Esc`.
- Pastikan setiap entri memiliki tanggal berlaku yang benar.

Troubleshooting:
- Jika file lampiran tidak muncul, cek ukuran file dan ekstensi yang diizinkan.
