# WiFi Management — Panduan Admin

Fungsi: menambah dan mengelola daftar WiFi internal yang akan ditampilkan ke karyawan.

Fitur:
- Form tambah WiFi: SSID, password, lokasi, visibility (public/internal).
- Daftar WiFi: search, edit, delete, dan toggle visibility.
- Modal edit untuk mengubah detail SSID.

Langkah cepat:
1. Buka `WiFi Management`.
2. Klik `Tambah WiFi Baru` → isi SSID & password serta lokasi.
3. Simpan; entri muncul di daftar dan dapat dicari.

Tips:
- Gunakan nama SSID yang konsisten dan sertakan lokasi agar karyawan mudah menemukan.
- Batasi akses edit ke role admin saja.

Troubleshooting:
- Jika SSID tidak muncul di frontend, pastikan entri tidak di‑draft/unpublished dan refresh cache halaman publik.
